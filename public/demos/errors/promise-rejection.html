Unhandled Promise Rejection

This page contains an unhandled promise rejection - a failed async operation without a .catch() handler. Common in API calls and async/await code.

// This will cause an unhandled promise rejection
async function fetchData() {'{'}
  const response = await fetch('https://api.nonexistent.com/data');
  return response.json();
{'}'}

fetchData(); // No .catch() handler!

// Error: Failed to fetch
